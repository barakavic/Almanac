# v0.1.0 — Initial Release

**Released:** June 2026  
**Build:** `0.1.0+1`  
**Platform:** Android

---

## Overview

First working release of Almanac. Establishes the core reading experience, local library management, and Android intent integration. All data is stored locally on-device with no network dependency.

---

## Features Shipped

### Library & Organization
- Shelf screen displaying books grouped by genre, with a **Currently Reading** and **Unsorted Books** section
- Create and manage **genres** and **subgenres** via the Genre Management screen
- **Assign genre on import** — a genre picker sheet appears immediately after picking a PDF via the FAB
- **Drag and drop** books directly onto a genre shelf header to reassign them
- **Long-press** any book from any shelf section to open a unified action sheet (Open, Archive, Reassign, Cancel)
- **Reassign sheet** for fine-grained genre and subgenre reassignment

### Import & Intent Handling
- Import books via the **in-app file picker** (FAB on shelf screen)
- Import books from external apps via Android's **"Open With"** intent (`ACTION_VIEW`)
- Duplicate import prevention via destination path locking — same file cannot be imported twice simultaneously
- PDF-only filter enforced on all import paths

### PDF Reader
- Full PDF rendering via **Syncfusion Flutter PDF Viewer**
- **Reading progress saved automatically** — debounced SQLite write fires 2 seconds after the last page turn
- Progress saved as a safety net on **app lifecycle pause** and **back navigation**
- **Dark mode toggle** — inverts PDF colors for low-light reading
- **Quote sharing** — highlight text, tap Share Quote, generates a branded quote card image and shares it via the system share sheet
- Image cache bumped to **200MB** for smoother page rendering
- Image cache cleared on reader exit to immediately return memory to the OS

---

## Architecture

- **State management:** Riverpod (`ConsumerStatefulWidget`, `AsyncNotifierProvider`)
- **Database:** SQLite via `sqflite` with a single `DatabaseHelper` singleton
- **Repository pattern:** `BookRepository`, `GenreRepository`, `SubgenreRepository`
- **Modular shelf widgets:** `ShelfScreen` delegates rendering to `CurrentlyReadingSection`, `UnsortedBooksSection`, and `GenreBooksSection`

---

## Known Limitations

- Page rendering speed is bound by Syncfusion's Flutter canvas layer — large PDFs on low-end devices will render slower than native PDF apps
- Archive feature is UI-complete but the SQLite write is not yet wired up
- Grid view layout is a placeholder (`_isGridView` toggle exists but renders a "coming soon" message)
- No search or sort functionality yet
